Enhancements

1. More Interactive Navigation Bar and Table

The assignment requires that a menu should link to every page on your website. There are no other CSS requirements for the menu. Therefore, it is only necessary to have links at the top of your page. My top nav has a combination of html and CSS that changes background and font colour when hovering over it and when selected, and formats into a neat and clean button-like menu. This mainly required the CSS event hover and other more advanced CSS.

The table requires some data and differing background colours for the heading and data cells. I took it a step further by again incorporating the hover event again when hovering over the table. Also, I utilised tr:nth-child(even) to alternate row background colours.

I learned these from w3schools at: https://www.w3schools.com/howto/howto_js_topnav.asp & https://www.w3schools.com/css/css_table.asp

Navigation Bar Table

2. Radiobutton and Checkbox with Media Queries.

The assignment requires content to align with window resize. I used a media query to extend beyond basic fitting functionality to re-format the quiz radio buttons and checkboxes based on window size so that it fits into view in a way that makes more sense than horizontal.

This required only some code using @media.

I learned these from w3schools at: https://www.w3schools.com/cssref/css3_pr_mediaquery.asp
