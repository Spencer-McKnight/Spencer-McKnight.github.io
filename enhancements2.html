Enhancements 2 (Javascript)

1. Timer and late submission status

The timer is triggered on the loading of the quiz page. Rather than onload, it is triggered using an event listener. For a programmer to implement this feature, they must implement a count, as well as the use of an interval function. An interval iterates after a given amount of time in ms, which depletes the count. On zero, a programmer can cause an action. In this instance, I added a late status that is changed to true.

I utilised the timer example displayed on codegrepper to assist:

'Foolish Ferret' 2020, Add Countdown Timer to Javascript Quiz, Grepper, viewed 25 September 2020, <https://www.codegrepper.com/code-examples/javascript/add+countdown+timer+to+javascript+quiz>

2. Prefilling User Information on re-attempt.

The prefilling of student information is similarly triggered on loading of the quiz page with an eventListener. To implement this feature, a programmer first checks if there is a name stored in session storage. If so, it gets first name, family name and student ID from session storage and assigns their values to the relevant textboxes.

I did not utilise external code. This enhancement can only be seen after a complete attempt of the quiz.
